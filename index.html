01 — Concept

プロジェクトの全体像

自律成長する営業インテリジェンス

本システムは、単なるマニュアル検索機ではありません。


現場の「商談録音」や「日報」から、 勝てる理由（成功法則）と負ける理由（課題）をAIが自動で抽出。 教材やスクリプトへリアルタイムに変換し続ける、 自律成長型のプラットフォームです。

営業組織の知見を蓄積 → 分析 → 変換 → 共有するサイクルを自動化し、
組織全体の営業力を継続的に底上げします。

02 — Input Data

構築に必要なインプットデータ

4つのデータソースが、知能基盤の精度を決定します。

Theory

商材・キャンペーン資料

PDF/PPT形式の商品資料。正確な商品知識、最新のプラン、約款をAIの知識基盤として投入。

Technique

トップ営業の商談録音

成果を出している人の「話し方」「切り返し」「間の取り方」を音声から自動解析。

Experience

過去の日報・Q&A履歴

現場で実際に起きたトラブル、顧客からの鋭い質問、それへの回答をテキストで蓄積。

Structure

顧客ターゲットリスト

どの業種・属性の顧客に、どの商材を売るべきかの定義。戦略の骨格を構築。

03 — Core Features

目的別5大機能

選択した商材ごとに、以下の機能が連動し「ナレッジ循環サイクル」を形成します。

01
FEATURE

Quick-FAQ

基本知識チャットbot

商材資料に基づいた正確な回答を即時提供。 後述の「深掘り」で得られた新しい知見が、自動で「最新FAQ」としてここに反映されます。 常に最新の情報にアップデートされ続ける、生きたナレッジベースです。

即時回答 資料ベース 自動更新
02
FEATURE

Closing-AI

営業ノウハウチャットbot

ターゲット別のスクリプト生成、反論処理を支援。 録音分析で判明した「今、最も刺さるフレーズ」が自動で「成功事例集」として格上げされ、 スクリプトに組み込まれます。組織の勝ちパターンが常に進化し続けます。

スクリプト生成 反論処理 成功事例自動反映
03
FEATURE

Daily-Insight

日報・深掘りコーチング

入力された日報に対し、AIが「なぜ失注した？」「顧客の反応のどの部分に違和感があった？」と 執拗に深掘り質問を実施。この対話から得られた「現場の教訓」を自動で整理し、 「成功・失敗事例集」としてデータベースに格上げ保存。組織全体に共有します。

AI深掘り質問 コーチング 教訓の自動ナレッジ化
04
FEATURE

Roleplay-Studio

AIロールプレイング特訓モード

失注分析の結果（個人の弱点）をAIが学習。 「あなたが昨日、失注したのと同じパターンの顧客」をAIが演じ、 克服できるまで何度でも特訓。実践に最も近いシミュレーション環境を提供します。

弱点特化型 実践シミュレーション 個別最適化
05
FEATURE

Lost-Analysis

録音・失注分析

音声から「声のトーン」「自信のなさ」「説明不足」を特定。 全体の失注理由を統計化し、管理画面へ「今、組織としてこの知識が不足している」と アラートを出力。個人と組織、両方の課題を可視化します。

音声解析 失注理由の統計化 組織課題アラート

06 — Security

代理店向けリスクヘッジ

情報資産を守る多層防御を標準実装。

  • 01
    権限階層制御
    代理店ランクにより閲覧可能な情報レベルを制限。 機密性の高い営業ノウハウやスクリプトを段階的に開示します。
  • 02
    透かし埋め込み
    表示される全コンテンツにユーザーIDの電子透かしを不可視レベルで埋め込み。 流出元の特定を即座に可能にします。
  • 03
    コピペ検知・即時停止
    不正なコピー&ペースト操作を検知した場合、 該当アカウントを即時停止。情報流出を未然に防ぎます。
