# AGI-Sales Driver 非機能要件

## 1. セキュリティ

### 1.1 データ分離

| 要件 | 内容 |
|------|------|
| テナント分離 | テナント間でデータは完全に分離。クロステナントアクセス不可 |
| ロールベースアクセス | admin / field の2ロールで権限制御 |
| API認証 | 全APIエンドポイントに認証トークン必須 |

### 1.2 情報流出防止

| 要件 | 内容 |
|------|------|
| 権限分離 | テナント間のデータ分離、ロールベースアクセス制御 |
| ID透かし | エクスポートドキュメントにユーザーIDの電子透かし埋め込み |
| 大量コピー検知 | 一定閾値を超えるデータコピー・ダウンロードでアラート |
| 監査ログ | 全操作を記録（who, what, when, where） |
| セッション管理 | 無操作30分でセッションタイムアウト |

### 1.3 外部連携セキュリティ

| 要件 | 内容 |
|------|------|
| Google Drive | OAuth 2.0 による認可。管理者のみ連携可能 |
| 音声データ | アップロード時の暗号化、保存時の暗号化（at-rest） |
| AI/LLM API | APIキーのサーバーサイド管理、クライアント非露出 |
