# AGI-Sales Driver デザイン要件

## 1. 美的方向性

| 項目 | 方針 |
|------|------|
| トーン | プロフェッショナル × ダーク × ゴールドアクセント |
| ムード | 高機能な「コックピット」感。営業戦闘ツール |
| 参照 | 現行のダーク/ゴールドテーマを継承・洗練 |
| 差別化 | 営業の現場感（速度・実用性）とAI知能感の融合 |

---

## 2. カラーパレット（現行テーマ継承）

| 用途 | カラー |
|------|--------|
| Background | `#0a0a0a` ダークベース |
| Surface | `#1a1a1a` カード・コンテナ |
| Border | `#2a2a2a` 区切り線 |
| Gold Accent | `#c8a96e` メインアクセント |
| Gold Hover | `#d4b87a` ホバー状態 |
| Text Primary | `#ffffff` |
| Text Secondary | `#9ca3af` |
| Success | `#22c55e` 成約・達成 |
| Warning | `#f59e0b` 注意・検討中 |
| Danger | `#ef4444` 失注・アラート |

---

## 3. タイポグラフィ

| 要素 | フォント | サイズ |
|------|---------|--------|
| 見出し | Noto Sans JP 700 | 1.5rem - 2rem |
| 本文 | Noto Sans JP 400 | 0.875rem - 1rem |
| データ | Noto Sans JP 500 tabular-nums | 0.875rem |
| コード・数値 | monospace系 | 0.8rem |

---

## 4. レイアウト方針

| 項目 | 方針 |
|------|------|
| 構造 | 左サイドバー + メインコンテンツ + 右サイドパネル（コンテキスト情報） |
| レスポンシブ | PC優先（営業現場はPC利用が主）、日報のみモバイル対応 |
| 密度 | 情報密度高め（営業は効率重視） |
| カード | ダークサーフェス + 1px border + 微ホバーエフェクト |

---

## 5. コンポーネント方針

| 項目 | 方針 |
|------|------|
| ベースライブラリ | Tailwind CSS（現行） |
| アイコン | lucide-react（現行） |
| チャットUI | 左右配置型チャットバブル（現行） |
| フォーム | ダークテーマ対応のカスタムフォーム |
| グラフ・チャート | recharts or Chart.js（管理画面用） |

---

## 6. レスポンシブ方針

| ブレークポイント | 対応 |
|----------------|------|
| Desktop（1024px+） | フル機能。3カラムレイアウト |
| Tablet（768px-1023px） | 2カラム。右パネル折りたたみ |
| Mobile（<768px） | 1カラム。日報入力・FAQ閲覧に最適化 |

## 8. アニメーション・インタラクション

| 項目 | 方針 |
|------|------|
| トランジション | 150-200ms、ease-out |
| ローディング | スケルトンスクリーン or スピナー |
| チャット | タイピングインジケーター（現行） |
| アラート | Human Touch Focusのアラートは赤パルスで視認性確保 |
| 抑制 | 過度なアニメーション禁止。営業ツールは速度優先 |
